# Superstore Sales Dashboard

An end-to-end sales analysis project built on a real-world retail dataset 
containing 5,010 orders and $1.09M in total revenue across 4 US regions.

## Dashboard Preview
![Superstore Sales Dashboard](dashboard.png)

## Features
- Interactive dashboard with 4 chart types (bar, line, horizontal bar, grouped bar)
- 3 dynamic slicers — Region, Category, Order Date
- Sales analysis by Region, Category, Sub-Category and Top 10 Customers
- Monthly sales trend visualization
- Statistical analysis — mean, median, mode, standard deviation, correlation

## Tools Used
- Microsoft Excel
- Power Query (data cleaning & transformation)
- Pivot Tables
- Data Visualization & Dashboard Design

## Key Insights
- Technology was the top-performing category ($380K in sales)
- West region led all regions with $332K in revenue
- Adrian Barton was the top customer by sales value ($12.1K)
- Total of 5,010 orders across 4 regions: West, East, Central, South
